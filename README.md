# Lianor reimagined static site v4

This folder contains the complete static website for deployment on GitHub Pages.

## Main files

- `index.html`: neutral bilingual gateway page
- `formazione-inglese.html`: Italian sales page for hospitality English
- `project-coordination.html`: English sales page for international project coordination
- `styles.css`: shared visual system and responsive layout
- `script.js`: fixed-header behavior, mobile navigation, progress line and reveal effects
- `assets/nobg.svg`: primary logo asset

Upload the contents of this folder to the root of the GitHub Pages repository. Keep the existing custom-domain configuration and confirm that the repository still contains the correct `CNAME` file if GitHub manages the domain through one.

Visual refinement v4: original navy logo in a white masthead medallion, white contact button, tighter reporting and hospitality panels, non-overlapping project hero note, US spelling throughout the English copy and expanded SEO copy for renovations, corporate events and conferences.
